Project Introduction

TeamUp is our 24-hour coding Hackathon project by team Code Ravana. This tool addresses the needs of remote teams for task assignment, user management, file sharing, and more. Our project was selected among the top 10 teams in the competition, showcasing real-time collaboration features using PHP and MySQL for data persistence.

We developed user registration & authentication, project creation, team collaboration modules, file sharing, and a dynamic calendar, ensuring a centralized workspace for remote teams.

Tech Stack & Usages

  • Language: PHP for server-side logic and templating.
  • Database: MySQL for storing user, task, and project data.
  • Front-End: HTML, CSS, JavaScript (Bootstrap for styling).
  • Version Control: Git & GitHub for collaborative development.
  • Deployment: The app can be hosted on any LAMP stack or similar environment.

2. User Registration (register.php)

{`prepare($sql);
    $stmt->bindParam(':username', $username);

This code handles user registration with password hashing for security, ensuring all credentials are stored safely in the database.

3. Task Creation (create_task.php)

{`prepare($sql);
    $stmt->execute([

This snippet demonstrates how tasks are created and assigned to specific team members, linking them to projects within the database.

4. Calendar Integration (calendar.php)

{`prepare($sql);

The calendar.php file retrieves task deadlines as calendar events. These events are returned in JSON format, making it easy to integrate with popular JavaScript calendar libraries.
